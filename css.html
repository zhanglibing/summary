IE兼容性问题

  1. IE9-10不支持select要加select::-ms-expand { display: none; }
  2. select在苹果手机中要加background：transparent 否则不兼容
  3. IE8-IE9都不支持transform中的属性
  4. 不支持nth-child(n)

  5. IE8兼容圆角和阴影方法

    将PIE.htx放入js文件夹中
    在css中
    -moz-border-radius:22px;
    -webkit-border-radius:22px;
    border-radius: 22px;
    behavior: url(js/PIE.htc);//相对于页面的路径

Flex布局

pc端IE兼容性差、手机端可以使用

阮一峰语法篇

  • 兼容方案：父级flex布局

                    
  Flex 布局以后，子元素的 float、clear和vertical-align属性将失效。
  容器的6个属性:
  flex-direction
  flex-wrap
  flex-flow
  justify-content
  align-items
  align-content

  子元素的6个属性
  order：number 定义项目的排列顺序。数值越小，排列越靠前，默认为0
  flex-grow
  flex-shrink：number 定义了项目的缩小比例  0的时候不缩放
  flex-basis
  flex
  align-self
                
            
                    
 display: -webkit-box;   /* OLD - iOS 6-, Safari 3.1-6 */
 display: -moz-box;  /* OLD - Firefox 19- H5不用考虑 */
 display: -ms-flexbox; /* TWEENER IE 10 */
 display: flex; /* NEW, Spec - Opera 12.1, Firefox 20+ */

  • 水平垂直居中

                    
.box {
  display: flex;
  justify-content:center; 水平方向
  align-items:center;  垂直方向
}

  • 平均分布

                    
  .parent{
    display: flex;
  }
  .child {
    flex: 1;
  }

  • 有一个宽度固定

                    
  .box {
    display: flex;
    align-items: center;
  }
  .other {
      width:150px;
      flex-shrink:0; //设置缩放为0
  }
  .item {
      flex: 1;
  }

  • 固定底栏

                    
 body {
    display: flex;
    min-height:100vh;
    flex-direction:column;
  }
  .content {
    flex: 1;
  }
