IE兼容性问题

  1. IE9-10不支持select要加select::-ms-expand { display: none; }
  2. select在苹果手机中要加background：transparent 否则不兼容
  3. IE8-IE9都不支持transform中的属性
  4. 不支持nth-child(n)

  5. IE8兼容圆角和阴影方法

    将PIE.htx放入js文件夹中
    在css中
    -moz-border-radius:22px;
    -webkit-border-radius:22px;
    border-radius: 22px;
    behavior: url(js/PIE.htc);//相对于页面的路径
