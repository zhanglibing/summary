JQUERY

  1. slides.filter('.active');筛选选中状态的元素
  2. 合并两个数组 $.merge( [0,1,2], [2,3,4] )结果:[0,1,2,2,3,4]
  3. 去掉字符串起始和结尾的空格$.trim(str);
  4. siblings()所有同级元素
    prev() 同级元素的前一个元素 next()之后一个
    prevAll()同级元素前所有的元素 nextAll()之后所有
  5. remove() - 删除被选元素（及其子元素） empty() - 从被选元素中删除子元素
  6. 求尺寸 width() height()
    innerWidth() innerHeight() 内填充
    outerWidth() outerHeight() 边框之内
    outerWidth(true) outerHeight(true) 边距之内
  7. 同时绑定两个事件类型
    $('#foo').bind('mouseenter mouseleave', function() { $(this).toggleClass('entered'); });
  8. 同时绑定多个事件类型 
    
   $("button").bind({
      click:function(){$("p").slideToggle();},
      mouseover:function(){$("body").css("background-color","red");},
      mouseout:function(){$("body").css("background-color","#FFFFFF");}
   });
  9. hover事件切换 
    
  $("aa").hover(function () {over},function () {out})
  10. each遍历
    
  (1)DOM元素 $("div").each(function (index, domEle) { }
  (2)数组 $.each( [0,1,2], function(i, n){});
  (3)对象 $.each( {{ name: "John", lang: "JS" },...}, function(i, n){
      alert( "Name: " + i + ", Value: " + n );
   });

    $(this).removeClass('active').dequeue();
  });自己测试最小150ms
