  • 创建vue项目

                    
 npm install -g vue-cli(如果电脑已安装过直接跳过)vue init webpack myapp(项目名称)
 都选择no,项目生成好后cd到当前项目
 npm install，安装所有依赖到node_modules目录中 （npm版本过低用npm install npm -g升级）
 npm run dev运行项目
                
            
        
 1.数据渲染： v-text、v-html、 {{}}
 <p>{{a}}</p>
 <p v-text="a"> </p>
 <p v-html="a"> </p>
2.控制模块隐藏 ：v-if、 v-show
 <p v-if="isShow"> </p>
 <p v-show="isShow"></p>
3.渲染循环列表
   <ul>
       <li v-for='item in items' >
            <p  v-text='item.lable'></p>
      </li>
   </ul>
4.事件绑定
<button v-on:click="doThis"><button> 完整语法
<button @click="doThis"><button> 一般使用这种简写方法
属性绑定:（herf、src、id、name、class...）v-bind   可简写 ：
<img  v-bind:src="imageSrc"> 
<div :class="{red:isRed}"><div> 
<div :class="[classA,classB]"><div> 
<div :class="[classA,{classB:isB,classC:isC}]"><div> 
5. Vue.js 都提供了完全的 JavaScript 表达式支持。
 {{ number + 1 }}
 {{ ok ? 'YES' : 'NO' }}
 {{ message.split('').reverse().join('') }}
 有个限制就是，每个绑定都只能包含单个表达式
 <div v-bind:id="'list-' + id"><div>
