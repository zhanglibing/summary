手机端测试方法

  1. ipconfig（window系统） ifconfig（苹果系统）
  2. 找到IPv4 地址 . . . : 192.168.31.180
  3. 将localhost替换为192.168.31.180
  4. 找到IPv4 地址: 192.168...
  5. 手机直接搜索网址进行测试 或者 进入网址cli.im 打开草料二维码 将网址生成二维码进行扫码测试

创建vue项目

  1. npm install -g vue-cli(如果电脑已安装过直接跳过)
  2. vue init webpack myapp(项目名称)
  3. 都选择no,项目生成好后cd到当前项目
  4. npm install，安装所有依赖到node_modules目录中 （npm版本过低用npm install npm -g升级）
  5. npm run dev运行项目
